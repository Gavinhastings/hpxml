Use Case

Contractor examines existing HVAC system (Gas Furnace and AC) and recommends/sells a new system (Gas Furnace and AC)

Minimum Data:

  • MaintainBuilding dataset which contains information about the HVAC systems currently
  • MaintainBuilding dataset which contains information about the new HVAC systems

Optional Data:

  • MaintainCustomer dataset with basic customer information
  • MaintainProject dataset which contains information about cost of installation and hardware
  • MaintainUtility dataset with customer's utility information

Data Set

  • Original equipment
  • New Equipment
  • Customer
  • Utilities
  • Project
