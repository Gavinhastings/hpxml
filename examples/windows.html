Use Case

Contractor replaces windows in a house.

Minimum Data:

  • MaintainBuilding dataset which contains information about the currently installed windows
  • MaintainBuilding dataset which contains information about the newly installed windows

Optional Data:

  • MaintainCustomer dataset with basic customer information
  • MaintainProject dataset which contains information about cost of installation and hardware

Data Set

  • Original equipment
  • New Equipment
  • Customer
  • Project
